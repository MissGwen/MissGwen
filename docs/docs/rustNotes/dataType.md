### 数据类型

#### 标量类型 （4种）

1. 整数类型 
   - 整数类型没有小数部分

   - 无符号 `u` 开头 😑

   - 有符号 `i` 开头 😑

     | length | signed | unsigned |
     | ------ | ------ | -------- |
     | 8-bit  | i8     | u8       |
     | 16-bit | i16    | u16      |
     | 32-bit | i32    | u32      |
     | 64-bit | i64    | u64      |
     | arch   | isize  | usize    |

     | 进制          | 表示方式              |
     | ------------- | --------------------- |
     | 10            | 98_222  '_'增强可读性 |
     | 16            | 0xff                  |
     | 8             | 0o77                  |
     | 2             | 0b1111_0000           |
     | Byte(u8 only) | b 'A'                 |

   - 整数溢出

     - u8类型范围0-255 如果把变量值设为256 那么编译时 会检查溢出 程序在运行时就会panic
     - 发布模式下 就不会检查导致panic的整数溢出
     - 如果溢出就会执行 “环绕” 操作 ```256``` 会变为 ```0```

2. 浮点类型

   - f32 32位数 单精度
   - f64 64位数 双精度

3. 布尔类型

4. 字符类型
